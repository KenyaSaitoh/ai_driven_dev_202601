### 記号
- **[ ]** 未開始

### 依存関係ルール
1. [P]マーク以外のタスクは上から下へ順次実行
2. [P]タスクは同一セクション内の他の[P]タスクと同時実行可能
3. [→]マークのタスクは直前のタスクの完了が必要
4. サブタスクは親タスクの依存関係を継承
